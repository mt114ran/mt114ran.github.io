13:T1921,<h1 id="webアクセシビリティ実装ガイドすべての人に使いやすいサイトを">Webアクセシビリティ実装ガイド：すべての人に使いやすいサイトを</h1>
<p>アクセシブルなWebサイトを構築するための実践的なガイドラインを紹介します。</p>
<h2 id="なぜアクセシビリティが重要か">なぜアクセシビリティが重要か</h2>
<ul>
<li><strong>包括性</strong>: すべてのユーザーがサービスを利用できる</li>
<li><strong>法的要件</strong>: 多くの国で法的要件となっている</li>
<li><strong>SEO向上</strong>: アクセシブルなサイトは検索エンジンにも優しい</li>
<li><strong>UX改善</strong>: すべてのユーザーにとって使いやすくなる</li>
</ul>
<h2 id="1-セマンティックhtml">1. セマンティックHTML</h2>
<p>適切なHTML要素を使用することが基本です。</p>

&#x3C;!-- 良い例 -->

</code></pre>

  const modalRef = useRef&#x3C;HTMLDivElement>(null);

      modalRef.current?.focus();

<h2 id="4-色のコントラスト">4. 色のコントラスト</h2>
<p>WCAG 2.1のガイドラインに従います：</p>
<ul>
<li><strong>通常テキスト</strong>: 最小コントラスト比 4.5:1</li>
<li><strong>大きなテキスト</strong> (18pt以上): 最小コントラスト比 3:1</li>
<li><strong>AAA準拠</strong>: 通常テキスト 7:1、大きなテキスト 4.5:1</li>
</ul>
<pre><code class="language-css">/* 良いコントラストの例 */
.text-primary {
  color: #1a1a1a;  /* 濃いグレー */
  background-color: #ffffff;  /* 白 */
}

/* 色だけに依存しない */
.error {
  color: #d32f2f;
  border: 2px solid #d32f2f;
  background-color: #ffebee;
}
</code></pre>
